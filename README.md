# OpenClaw 像素房间

一个可视化展示 OpenClaw AI Agent 工作状态的像素风格网页。

🎮 **Live Demo**: https://ddddk8469-stack.github.io/pixel-agent-room/

## 功能特点

- 🏠 像素风格的房间场景
- 💻 三个工作区域：电脑桌（写代码）、茶几（聊天）、书架（查资料）
- 🦞 像素小人根据工作状态在不同区域移动
- 💭 思考气泡显示当前活动
- 📊 实时状态面板

## 场景说明

| 区域 | 活动 |
|------|------|
| 电脑桌 💻 | 写代码、操控电脑 |
| 茶几 💬 | 与用户聊天对话 |
| 书架 📚 | 查询资料、搜索信息 |
